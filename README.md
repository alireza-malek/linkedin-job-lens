# LinkedIn Job Lens

A Chrome extension that helps job seekers find LinkedIn job postings with visa sponsorship or relocation assistance. The extension automatically scans job listings, highlights relevant keywords, and saves matching jobs to a convenient list.

## ✨ Features

- **Auto-Scan**: Automatically scans LinkedIn job listings for visa sponsorship keywords
- **Custom Keywords**: Add your own keywords to search for specific terms
- **Smart Highlighting**: Highlights visa/relocation keywords in job descriptions
- **Job Saving**: Save scanned jobs with match details to a searchable list
- **Export/Import Saved Jobs**: Copy job descriptions or export saved jobs
- **Badge System**: Visual badges on job cards showing match status
- **Full List Scanning**: Automatically scan entire job search result pages

## 🚀 Installation

1. Go to the [Releases](https://github.com/alireza-malek/linkedin-job-lens/releases) page
2. Download the latest `linkedin-job-lens-vX.X.X.zip` file and extract it
3. Open Chrome and navigate to `chrome://extensions/`
4. Enable "Developer mode" (toggle in the top right) and click "Load unpacked"
5. Select the extracted folder. The extension is now installed!

## 📝 Permissions

This extension requires the following permissions:

- `activeTab`: To interact with LinkedIn pages
- `scripting`: To inject content scripts
- `sidePanel`: To display the extension panel
- `storage`: To save your scanned jobs and settings
- `tabs`: To track active tabs
- `downloads`: To export saved jobs
